# Saforaai - Compassionate Elderly Care

## Project info

Building a purpose-driven elderly care organisation delivering high-quality, reliable care with dignity, speed, and compassion in Stratford-upon-Avon, UK.

## How can I edit this code?

Follow these steps to work locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd saforaai-care-showcase

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

The project can be deployed to any static hosting provider like Vercel, Netlify, or Firebase Hosting.

```sh
npm run build
```
